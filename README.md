# Loan Prediction Model

Made by : [Sagara Biru Wilantara](https://www.linkedin.com/in/sagara-biru)

---

## Introduction

This project focuses on making machine learning model using 5 models (KNN, SVM, Decision Tree, Random Forest, and Ada Boost) for predicting loan approval.

---

## Project Objective

The purpose of this analysis is to help financial institutions make better decisions in the loan approval process by creating the best model that can predict whether an applicant is approved or not for a loan.

---

## Data Source

The dataset used in this project from kaggle [link](https://www.kaggle.com/datasets/ninzaami/loan-predication).
